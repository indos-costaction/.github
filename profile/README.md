# Improving Neuroimaging Data for Sharing (INDoS)

[COST Action CA24161](https://www.cost.eu/actions/CA24161/) brings together researchers and stakeholders across Europe to improve the sharing of human neuroimaging data by leveraging consortium expertise to establish standards, tools, and training, thereby enhancing its findability, accessibility, interoperability, reusability, and ethical compliance.

GitHub is used for the project's website and for managing collaborative documentation and community resources.
